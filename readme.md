# Projeto de Otimização Lighthouse

## Descrição
Projeto que visa aplicar técnicas Lighthouse do zero para validação de metodologia de performance e qualidade de aplicações web.

## Métricas Lighthouse

### Core Web Vitals
- **LCP (Largest Contentful Paint)**: Mede o tempo de carregamento do maior elemento visível
- **FID (First Input Delay)**: Avalia a resposta da página à primeira interação do usuário
- **CLS (Cumulative Layout Shift)**: Quantifica mudanças inesperadas de layout durante o carregamento
- **TBT (Total Blocking Time)**: Soma de todos os períodos entre "FCP" e "Tempo para interação da página", quando a duração da tarefa ultrapassa 50 ms, expressa em milésimos de segundo.

### Outras Métricas
- **Performance**: Velocidade de carregamento e execução
- **Accessibility**: Conformidade com padrões de acessibilidade
- **Best Practices**: Seguir práticas recomendadas de desenvolvimento
- **SEO**: Otimização para mecanismos de busca
